## <u>Teamate</u> Project Background
- ### 필요성
  - 운전 중 스마트폰 조작은 사고 위험을 높이며, 음성 명령을 통한 차량 제어 필요성이 증가하고 있음
  - 차량 내부의 공기질(특히 CO₂ 농도)에 따라 졸음운전 위험이 높아질 수 있음
  - 운전자 중심의 맞춤형 음성 서비스와 실내 환경 관리 시스템 필요
  - 기존 상담원 기반 서비스를 AI로 대체하여 비용 절감 및 즉각적인 응답 제공
  - 추후 차량 제조사와 관계없이 사용가능한 범용적인 AI 비서 시스템 구축
    
- ### 기존 해결책의 문제점
  - 차량 제조사별 전용 AI 시스템은 특정 차량 모델에만 적용되어 범용성이 부족함
  - 고객센터 상담원 기반 시스템은 운영 비용이 높고, 응답 속도가 느려 즉각적인 대응이 어려움
  - 커스터마이징이 불가능하고, 기능 확장성 부족
  - 기존 음성 서비스는 공기 같은 차량 내부 환경 데이터를 활용하지 못해, 운전자 안전 관리 기능이 제한적임
  
## System Design
  - ### System Requirements
    - Raspberry Pi 5 기반 제어 보드
    - 음성 인식 모듈 및 미세먼 센서 연결
    - React + Tailwind CSS 기반의 사용자 UI (LCD 패널 출력)
    - 데이터베이스를 활용해 차종별 질문 저장
    - OpenAI API를 활용한 자연어 처리 및 응답 생성
